# SWEX RTA

This plugin will allow you to synchronize your matches with [rtapicks.info](https://www.rtapicks.info).

## Installation

1. Download the [latest release](https://github.com/chinleung/sw-exporter-rta/releases/latest/).
2. Move the **rta.asar** into the plugins folder of [SWEX](http://github.com/xzandro/sw-exporter).
